בתיקייה זו מופיע הקוד שכתבתי לצורך שאלה 1.
הקוד פותר את בעיית MDP הנידונה, עבור ערך r שניתן כארגומנט-שורת-פקודה.
התוכנית הראשית נמצאת בקובץ gridMDP.py.

כדי להריץ את התוכנית עבור ערך r מסויים, הרץ/י את הפקודה:
python gridMDP.py r
(הקוד נכתב עבור פייתון 3.7).

הפלט כולל שלוש "מפות":
> מפה של ערכי התגמול בכל נקודה על המטריצה.
> מפה של ערכי התועלת שחושבו ע"י אלגוריתם Value Iteration.
> מדיניות הפעולות האופטימלית שחושבה בהתאם לתועלות שחושבו ע"י Value Iteration.

הערה קטנה:
מכיוון שהעיקר בשאלה הוא לנתח את המדיניות האופטימליות שחושבה (ולא המימוש עצמו),
לא השקעתי הרבה בתיעוד (אף על פי שיש תיעוד בכמות מכובדת), ויכולתי לשפר את האיכות
של הקוד (למשל ב-direction.py). פשוט זה לקח לי יותר מדי זמן, והרגיש לי מיותר "לתת
את כל כולי" כשזה בכלל לא מה שנדרש מאיתנו בשאלה (כשאפשר בכלל לערוך סימולציה
ידנית "על דף" במקום, למשל).
בכל מקרה, האלגוריתם עובד בצורה נכונה (עשיתי לא מעט בדיקות).
